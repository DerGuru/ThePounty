# Übersetzungs-Glossar DE (Buch 1 → "10.1 - Disarming Courtesy/Deutsch")

Konsistente deutsche Entsprechungen wiederkehrender Wendungen (Buch-1-Übertragung, Juni 2026).
Bei Buch-2-Übersetzung beibehalten.

## Aldric-Stimme / Schlüsselwendungen
- "He respected / admired that" → variieren, NICHT immer "Das achtete er": **Das imponierte ihm. / Das rechnete er ihnen an. / Das schätzte er. / Das nötigte ihm Respekt ab.**
- "filed it / filed it away" → **ablegen** (er legte es ab / legte es zum Rest ab).
- "the extra beat" / "held the extra beat" → **den zusätzlichen Schlag (halten)**.
- "many happy returns (of the day)" → **alles Gute zum Geburtstag** (nachträglich: "nachträglich alles Gute").
- "a beat late / half a beat late" (Aldrics Lächeln) → **einen Schlag zu spät / eine halbe Sekunde zu spät**.
- "I'll see to it" (flache Arbeitsstimme) → **Ich kümmere mich darum.**
- "the small bow that was not a bow" → **die kleine Verbeugung, die keine Verbeugung war**.
- "It needed doing." → **Es musste getan werden.**
- promise: "a word given was permanent" → **ein gegebenes Wort war endgültig**; "I'll do" (knappes Versprechen) → **Ich tue es.**

## Figuren / feste Begriffe
- Officer Kim: **Officer Kim** (nicht "Polizistin Kim"); "cruiser" → **Streifenwagen**; "dispatch" → **Leitstelle**; Funk: "twelve" → **Zwölf**.
- "the watcher" (Gerald/Calvey-Kap.) → **der Beobachter**.
- "the cul-de-sac" → **der Wendekreis**; "the playground end / east end" → **Spielplatzende / Ostende**.
- "porch" → **Veranda**; "screen door" → **Fliegengittertür**; "mailbox" → **Briefkasten**; "curb" → **Bordstein**.
- "HOA" → **HOA** (nicht eindeutschen); "community center" → **Gemeindezentrum**.
- "recycling bin" → **Recyclingtonne**; "trash" → **Müll/Restmüll**.
- Gerald-Notizbücher: **das blaue / schwarze Notizbuch**; "account/ledger" Metaphorik → **Konto / aufgehen / verbuchen / Posten / abschließen**.
- Maria "I did that" → **Das habe ich gemacht.**
- "channel locks" → **Wasserpumpenzange**; "slip nut" → **Überwurfmutter**; "spanner wrench" → **Hakenschlüssel**.

## Stil-Hinweise (Glättung)
- Englische Partizipial-/Gerundialketten auflösen, nicht 1:1 nachbauen.
- "tat das, was es tat / tat die Sache" (literal "did the thing") → konkretisieren oder natürlicher fassen.
- Cleft "Es war X, das/der ..." sparsam; oft besser Vorfeld-Umstellung.
- Buchtitel im Englischen lassen (*The Bird Way*, *A Tangle of Knots*); spanische Figuren­rede (Maria-Familie) bleibt spanisch.
- "nichts Besonderes", "das Richtige", "das Erste" → Großschreibung substantivierter Adjektive prüfen.
